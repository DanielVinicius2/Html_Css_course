O que é IPSec?

Protocolo de Segurança para Redes IP

Introdução

IPSec (Internet Protocol Security) é um conjunto de protocolos utilizado para proteger comunicações via IP através de autenticação e criptografia de cada pacote IP em uma sessão de comunicação.

Objetivos do IPSec

  • Confidencialidade dos dados (Criptografia)
  • Integridade dos dados (Verificação de alterações)
  • Autenticação das partes envolvidas
  • Proteção contra ataques de repetição

Componentes Principais

O IPSec é composto por dois principais protocolos:

  • AH (Authentication Header): Garante integridade e autenticação.
  • ESP (Encapsulating Security Payload): Fornece criptografia, integridade e autenticação.

Modos de Operação

O IPSec pode operar em dois modos:

  • Modo Transporte: Apenas o conteúdo do pacote IP é protegido.
  • Modo Túnel: O pacote IP inteiro é encapsulado e protegido. Usado em VPNs.

Aplicações Comuns

O IPSec é amplamente utilizado em:

  • Redes Privadas Virtuais (VPNs)
  • Segurança entre roteadores e gateways
  • Proteção de dados sensíveis em redes corporativas
